# Política de Privacidade (Modelo Inicial)

Este documento descreve práticas de coleta, uso e retenção de dados para o GestorMEI.

## Dados Coletados
- Dados de autenticação (e-mail e hash de senha).
- Dados de operação (produtos, vendas, estoque, auditoria de ações críticas).
- Metadados técnicos (IP e user-agent para auditoria e segurança).

## Finalidade
- Operação do sistema de gestão.
- Segurança da conta e trilha de auditoria.
- Suporte técnico e prevenção de fraude.

## Retenção
- Dados de negócio: mantidos enquanto a conta estiver ativa.
- Logs de auditoria: mantidos por política interna de segurança.

## Direitos do Titular
- Solicitar acesso, correção e exclusão de dados quando aplicável.

## Segurança
- Senhas armazenadas com hash.
- Controle de acesso por JWT.
- Rate limiting de login e auditoria de ações críticas.

## Contato
- Definir e-mail de privacidade antes de uso em produção.
